Данное расширение автоматически добавляет лейблы к продуктам в самом продукте, в категориях, в производителях, в акциях и поиске.

Также в модуле рекомендуемые.

Не трогает файлы магазина.

Автоматически добавляет лейбл "Новинка" для товаров добавленных меньше 14 дней назад.

Автоматически добавляет лейбл "Распродажа" для товаров со скидками.

Язык: en/ru.

Тестировалось на Opencart 2.0 и Opencart 2.1

Демо на сайте: http://svg.proitservice.in.ua/

Установка: 

Через меню "Модули" -> "Установка расширений" установить файл "labels_OC_2.ocmod.zip".

Обновить кэш модификаторов "Модули" -> "Модификаторы" Нажав на кнопку "Обновить"
